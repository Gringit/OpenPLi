Italian translation SystemTime plugin.

Source:

https://github.com/OpenPLi/enigma2-plugins/tree/master/systemtime/po

How to update SystemTime plugin language?

1) Copy the compiled italian "SystemTime.mo" file in /usr/lib/enigma2/python/Plugins/SystemPlugins/SystemTime/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

Italian translation VFDControl plugin.

Source:

https://github.com/OpenPLi/enigma2-plugins/tree/master/vfdcontrol/po

How to update VFDControl plugin language?

1) Copy the compiled italian "VfdControl.mo" file in /usr/lib/enigma2/python/Plugins/SystemPlugins/VfdControl/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

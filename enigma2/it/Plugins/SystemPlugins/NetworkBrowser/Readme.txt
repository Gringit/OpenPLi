Italian translation NetworkBrowser plugin.

Source:

https://github.com/OpenPLi/enigma2-plugins/tree/master/networkbrowser/po

How to update NetworkBrowser plugin language?

1) Copy the compiled italian "NetworkBrowser.mo" file in /usr/lib/enigma2/python/Plugins/SystemPlugins/NetworkBrowser/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

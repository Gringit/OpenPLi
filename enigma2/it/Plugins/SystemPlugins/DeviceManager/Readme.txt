Italian translation DeviceManager plugin.

How to update DeviceManager plugin language?

1) Copy the compiled italian "DeviceManager.mo" file in /usr/lib/enigma2/python/Plugins/SystemPlugins/DeviceManager/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

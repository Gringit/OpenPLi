Italian translation AutoBackup plugin.

Source:

https://github.com/E2OpenPlugins/e2openplugin-AutoBackup/tree/master/plugin

How to update AutoBackup plugin language?

1) Copy the compiled italian "AutoBackup.mo" file in /usr/lib/enigma2/python/Plugins/Extensions/AutoBackup/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

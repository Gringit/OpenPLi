Italian translation TranscodingSetup plugin.

Source:

https://github.com/OpenPLi/enigma2-plugins/tree/master/transcodingsetup/po

How to update TranscodingSetup plugin language?

1) Copy the compiled italian "TranscodingSetup.mo" file in /usr/lib/enigma2/python/Plugins/Extensions/TranscodingSetup/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

Italian translation MovieCut plugin.

Source:

https://github.com/OpenPLi/enigma2-plugins/tree/master/moviecut/po

How to update MovieCut plugin language?

1) Copy the compiled italian "MovieCut.mo" file in /usr/lib/enigma2/python/Plugins/Extensions/MovieCut/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

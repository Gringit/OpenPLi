Italian translation OpenWebif plugin.

Source:

https://github.com/E2OpenPlugins/e2openplugin-OpenWebif/tree/master/locale

How to update OpenWebif plugin language?

1) Copy the compiled italian "OpenWebif.mo" file in /usr/lib/enigma2/python/Plugins/Extensions/OpenWebif/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

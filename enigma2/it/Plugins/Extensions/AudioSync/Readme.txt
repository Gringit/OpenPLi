Italian translation AudioSync plugin.

How to update AudioSync plugin language?

1) Copy the compiled italian "AudioSync.mo" file in /usr/lib/enigma2/python/Plugins/Extensions/AudioSync/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

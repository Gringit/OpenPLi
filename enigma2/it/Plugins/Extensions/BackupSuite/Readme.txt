Italian translation BackupSuite plugin.

Source:

https://github.com/PLi-metas/BackupSuite/tree/master/plugin/locale/it/LC_MESSAGES

How to update BackupSuite plugin language?

1) Copy the compiled italian "BackupSuite.mo" file in /usr/lib/enigma2/python/Plugins/Extensions/BackupSuite/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2

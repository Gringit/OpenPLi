Italian translation EPGImport plugin.

Source:

https://github.com/OpenPLi/enigma2-plugin-extensions-epgimport/tree/master/src/EPGImport/locale

How to update EPGImport plugin language?

1) Copy the compiled italian "EPGImport.mo" file in /usr/lib/enigma2/python/Plugins/Extensions/EPGImport/locale/it/LC_MESSAGES using FileZilla
2) Restart Enigma2
